CASB_x002E_ProcesoAutenticacionService

SOAP Version: 1.1

Main class which you can use to connect to your web service

Important settings

  • Timeout - the timeout for network operations, in milliseconds. By default: 60 sec.
    Example:
                        
CASB_x002E_ProcesoAutenticacionService service = new CASB_x002E_ProcesoAutenticacionService(null,"https://portal.bytte.com.co/casb/YoPrestoPR/CASB.ProcesoAutenticacion/Service/ServicioAutenticacion.svc",30000);
service.ConsultaProcesoValidacionDocumento();
  • Url - the web service address. By default it is retrieved from the wsdl file
    Example:
                        
CASB_x002E_ProcesoAutenticacionService service = new CASB_x002E_ProcesoAutenticacionService(null,"https://portal.bytte.com.co/casb/YoPrestoPR/CASB.ProcesoAutenticacion/Service/ServicioAutenticacion.svc");
service.ConsultaProcesoValidacionDocumento();
  • httpHeaders - allows you to add custom headers to the http requests.
    Example:
                        
CASB_x002E_ProcesoAutenticacionService service = new CASB_x002E_ProcesoAutenticacionService();
service.httpHeaders.add( new HeaderProperty("Authorization","Basic QWxhZGRpbjpPcGVuU2VzYW1l"));
service.ConsultaProcesoValidacionDocumento();
  • Enable logging - if you enable logging, all requests/responses will be write to the console (for debugging purposes)
    Example:
                        
CASB_x002E_ProcesoAutenticacionService service = new CASB_x002E_ProcesoAutenticacionService();
service.enableLogging=true;
service.ConsultaProcesoValidacionDocumento();
  • createClassesForAny - if true, detect a strong types in any collections
    Example:
    service.createClassesForAny=true;
